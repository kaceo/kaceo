CAPH key navigation Sample for jQuery


CAPH key navigation?
Key Navigation is an essential utility to use TV's remote
  • Simple & Easy : Add only 'focusable' attribute
  • Customizable : Use various event handlers
  • Convenient : CAPH handles 4 direction keys internally
    • More details, click here!

This sample is packaged based on caph 3.0.1. You can try with arrows keys.


Let's try

Step 1 Include the caph-jquery
<script src="caph/3.0.1/caph-jquery.min.js"> </script>
Step 2 Add focusable at each div
<div focusable >
Step 3 Add event handler
controllerProvider.onFocused(function(event, originalEvent) {
    $(event.currentTarget).css({
        border: '3px solid red'
    });
});
<div focusable on-focused="focus($event)" >
